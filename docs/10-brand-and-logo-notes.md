# Brand & Logo Notes

* **Name:** NetCrab

* **Logo Concept:**

  * Simple line-art hermit crab.

  * Shell = retro desktop computer or CRT monitor.

  * One claw holding a small glowing data cube.

  * Shape must work in monochrome for small favicons.

* **Illustration Usage:**

  * Use more detailed, friendly version for hero art.

  * Keep UI chrome mostly clean; let the crab appear in key spots:

    * Dashboard empty states.

    * "Crab Notes" panel.
