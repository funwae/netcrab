# Docker Infrastructure

Dockerfiles and configuration for NetCrab services.

## Services

- **NetCrab Agent**: On-premises collector service
- **Ingestion API**: Cloud ingestion service
- **Stream Processors**: Sessionizer, incident-detector, etc.
- **AI Services**: ML clustering and LLM services
- **Application API**: Dashboard and marketplace API
- **Console**: Customer dashboard (Next.js)
- **Marketing**: Marketing site (Next.js static)

## Development

Use `docker-compose.yml` in the parent directory for local development with all infrastructure services.
