# @netcrab/electron

NetCrab SDK wrapper for Electron and desktop applications.

## Installation

```bash
npm install @netcrab/electron
```

## Usage

```javascript
import { initNetCrab } from '@netcrab/electron';

initNetCrab({
  orgId: 'your-org-id',
  productId: 'your-product-id',
  agentUrl: 'https://agent.yourcompany.local'
});
```
