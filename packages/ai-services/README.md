# NetCrab AI Services

ML and AI services for session clustering, task archetype mining, and LLM-generated "Crab Notes" insights.

## Services

- **Session Clustering**: Groups sessions by behavior patterns (KMeans/HDBSCAN)
- **Task Archetype Mining**: Discovers common user journey patterns
- **LLM Insight Generator**: Generates plain-language summaries and improvement ideas

## Features

- Session-level clustering (high-friction short sessions, long wandering, fast success)
- Navigation sequence analysis
- LLM-powered insight summaries with improvement recommendations
